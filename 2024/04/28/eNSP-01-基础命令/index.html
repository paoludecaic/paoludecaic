基础命令

介绍一些暂时学到的命令

ping命令

ping ip地址

例：

ping 192.168.1.1

路由

IP配置

ip add ip地址 子网掩码

例：

int g0/0/1
ip add 202.38.165.1 255.255.255.0

查看配置 dis

dis cu
dis vlan
dis this

撤销错误命令 undo

undo 命令

示例：
配置int g0/0/3为trunk类型，并通过vlan10 20

  1. 错误示例：
int g0/0/3
port link-type access
port default vlan 10
  1. 撤销操作
undo port default vlan
undo port link-type
  1. 正确命令
int e0/0/3
port link-type trunk
port trunk allow-pass vlan 10 20
