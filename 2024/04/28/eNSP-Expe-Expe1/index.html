实验一

路由器的简单配置

Router路由器的简单配置
配置ip地址与子网掩码

实验一共三台路由器与一台交换机
只简单的配置下路由器，交换机暂不配置

基础的命令模版

进入系统视图

system-view

也可简化为

sys

设置设备名称

sysname NewName

也可简化为

sysn NewName

关闭系统信息提示

undo info-center enable

也可简化为

un in en

接口配置模式

千兆网口0/0/0
interface GigabitEthernet 0/0/0
百兆网口0/0/0
interface Ethernet 0/0/0

也可简化为

千兆网口0/0/0
int g0/0/0
百兆网口0/0/0
int e0/0/0

配置命令

Router1

<RouterA> system-view  # 进入系统视图
[RouterA] sysname R1   # 设置设备名称为R1
[R1] interface GigabitEthernet 0/0/0( int g0/0/0  )   # 进入接口GigabitEthernet 0/0/0配置模式
[R1-GigabitEthernet0/0/0] ip address 202.38.165.1 255.255.255.0  # 配置IP地址及子网掩码
[R1-GigabitEthernet0/0/0] quit  # 退出接口配置模式
[R1] undo info-center enable  # 关闭系统信息提示
[R1] quit  # 退出系统视图
<R1> save  # 保存配置

Router2

<RouterA> system-view  # 进入系统视图
[RouterA] sysname R2   # 设置设备名称为R2
[R2] interface GigabitEthernet 0/0/0   # 进入接口GigabitEthernet 0/0/0配置模式
[R2-GigabitEthernet0/0/0] ip address 202.38.165.2 255.255.255.0  # 配置IP地址及子网掩码
[R2-GigabitEthernet0/0/0] quit  # 退出接口配置模式
[R2] undo info-center enable  # 关闭系统信息提示
[R2] quit  # 退出系统视图
<R2> save  # 保存配置

Router3

<RouterA> system-view  # 进入系统视图
[RouterA] sysname R3   # 设置设备名称为R3
[R3] interface GigabitEthernet 0/0/0   # 进入接口GigabitEthernet 0/0/0配置模式
[R3-GigabitEthernet0/0/0] ip address 202.38.165.3 255.255.255.0  # 配置IP地址及子网掩码
[R3-GigabitEthernet0/0/0] quit  # 退出接口配置模式
[R3] undo info-center enable  # 关闭系统信息提示
[R3] quit  # 退出系统视图
<R3> save  # 保存配置
