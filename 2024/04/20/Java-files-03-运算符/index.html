Java基础-3-运算符

运算符

1、算术运算符

加减 乘除模

+   加
-   减
*   乘
/   除
%   模

自增自减

i++     前自增
++i     后自增
i--     前自减
--i     后自减

2、赋值 运算符

=       赋值        int a = 100
+=      加等于      a+=10 <==> a=a+10
-=      减等于      a-=10 <==> a=a-10
*=      乘等于      a*=10 <==> a=a*10
/=      除等于      a/=10 <==> a=a/10
%=      模等于      a%=10 <==> a=a%10

3、比较 运算符

==      等于
!=      不等于
>       大于
>=      大于等于
<       小于
<=      小于等于
