Java基础-4-选择语句

选择语句

共3部分
两个语句结构和一个特殊三元运算符
三元运算符包含在 if 语句中理解

if 语句系

参考代码详见 代码 部分
参考代码适用于有一定基础的
博客部分提供的代码较为简单

if 语句

以取两个数设为最大值为例

int x=10;
int y=20;
int max = 0;
if (x > y) {
    max = x;
}
if (y > x) {
    max = y;
}

if … else 语句

以判断奇偶数为例

int x=sc.nextInt();
if(x%2==0){
    System.out.println("偶数");
}else{
    System.out.println("奇数");
}

if … else if … if 语句

以判断大小写字母为例

char ch=sc.next();

if(ch>='a'&&ch<='z'){
    System.out.println("小写");
}else if(ch>='A'&&ch<='Z'){
    System.out.println("大写");
}else{
    System.out.println("不是26个字母");
}

三元运算符 ?:

?: 是一个特殊的运算符
可以使用三元运算符来代替代码中简单的 if if…else if … else if … if 来简化代码
可以嵌套使用

例1：
int num=x>y>?x:y;

等价于

int num;
if(x>y){
    num=x;
}else{
    num=y;
}

例2：
int num=(x>y>?x:y)>z?(x>y>?x:y):z;

等价于

int num=x;
if(y>num){
    num=y;
}else if(z>num){
    num=z;
}
